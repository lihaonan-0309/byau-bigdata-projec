

## 项目描述



根据@daidaibg**(https://gitee.com/daidaibg/IofTV-Screen)**
开源框架基础上进行修改。前端界面，后端自主实现。
计算机设计参赛作品，前端使用IofTV-Screen大屏模板，后端为自主实现的深度学习算法。


本文件内容为系统的整体实现代码、算法训练和验证代码、训练、验证数据集、可视化数据集：
1.代码文件：
	前端代码.rar：系统前端实现代码；
	后端代码.py：后端实现代码；
	机器学习代码.py：机器学习模型训练代码；
	卷积神经网络代码：基础卷积神经网络训练代码；
	改进卷积神经网络代码：基础卷积神经网络代码基础之上改进模型训练代码；
	多任务学习代码：基础多任务学习训练代码；
	改进多任务学习代码：基础多任务学习代码之上改进后的多任务学习训练代码。
2.数据文件：
	训练数据：包含标签、特征的训练数据，截取100行样本展示；
	测试数据：用于测试系统功能的数据，共100行。
	20年间农用化肥施用量(国家统计局年度数据)：可视化大屏数据显示，数据来源国家统计局2023统计年鉴；
	主要农作物生长对主要养分需求量、土类分布情况、土种分布情况：可视化大屏数据显示，数据来源中国土壤数据库。
3.pth文件：model.zip文件下包含所有训练后模型文件。
	ph_model：ph含量训练后模型(以ph值为例)；
	mtl_model ：多任务学习训练后模型。
